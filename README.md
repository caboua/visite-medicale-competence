# VISITE MÉDICALE & COMPÉTENCE

Prototype PWA iPhone pour le suivi des aptitudes médicales du CIS Bouillante.

Installation iPhone : héberger le dossier, ouvrir avec Safari, puis Partager > Sur l’écran d’accueil.

Fonctions : tableau de bord, agents, filtres, priorités, analyse simple d’un mail collé, export JSON.
