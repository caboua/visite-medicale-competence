Tu analyses un mail ou un certificat médical SDIS. Extrais uniquement les agents dont le CIS/Service est CIS BOUILLANTE. Si le CIS est différent, répondre IGNORER.
Retourne un JSON strict avec : nom, prenom, matricule, cis, date_certificat, type_visite, avis_medical, inaptitude, type_inaptitude, restriction, activites_sportives, prochaine_visite, document, source.
Règles : Apte sans restriction => inaptitude Non. Apte avec restriction => restriction renseignée. Inapte temporaire/définitif/partiel => inaptitude Oui. Ne jamais intégrer un agent non Bouillante.
