VERSION V15 - SCAN OCR RENFORCE

Objectif : corriger l'erreur 'Tesseract OCR non charge'.

Cette version tente de charger le moteur OCR depuis plusieurs sources internet : jsDelivr, unpkg, cdnjs.
Elle desactive aussi l'ancien cache/service worker.

Utilisation :
1. Remplacer les fichiers sur GitHub.
2. Ouvrir : https://caboua.github.io/visite-medicale-competence/?reset=1&v=15
3. Faire CTRL + F5 sur ordinateur.
4. Aller dans Scan documents.
5. Selectionner le PDF ou l'image.
6. Corriger si necessaire puis appliquer a la base locale.
7. Telecharger agents.json et le remplacer sur GitHub pour rendre la mise a jour permanente.

Important : si le reseau bloque totalement les CDN, l'OCR local ne pourra pas fonctionner. Dans ce cas il faudra passer a la version Make + OpenAI Vision, qui analyse les PDF cote serveur.
