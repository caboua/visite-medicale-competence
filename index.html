Scanner un document téléchargé pour mettre la fiche à jour

Sélectionne un ou plusieurs certificats PDF ou images déjà téléchargés sur ton ordinateur ou ton iPhone. Le logiciel scanne chaque document, lit le contenu, détecte l’agent, vérifie Bouillante, puis prépare la mise à jour de la fiche.

Important : la lecture se fait dans le navigateur. Elle peut être plus lente sur iPhone. Si le document est flou ou manuscrit, le logiciel placera le résultat en À vérifier au lieu de modifier automatiquement la base.

Règle de sécurité Bouillante

Une fiche est modifiée automatiquement seulement si le document indique clairement CIS BOUILLANTE ou si le fichier contient explicitement Bouillante. Les autres centres sont exclus. Si le CIS n’est pas lisible, le logiciel affiche À vérifier manuellement.

Après mise à jour : clique sur Télécharger agents.json, puis remplace ce fichier sur GitHub pour que la base soit permanente.
