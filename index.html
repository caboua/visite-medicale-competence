Mise à jour depuis un certificat PDF

Version finale : plus aucun Tesseract, donc plus d’erreur OCR dans Chrome. Deux méthodes sont disponibles.

Méthode simple recommandée : tu m’envoies les PDF ici, je te donne le JSON de mise à jour ou un nouveau agents.json. Tu colles le JSON dans le bloc ci-dessous et tu appliques.

Méthode IA automatique : uniquement si tu as une URL Make/OpenAI Vision. Le PDF est envoyé à cette URL et l’IA renvoie les champs à appliquer.

1 — Mise à jour manuelle assistée

Choisis le PDF pour récupérer son nom de fichier, puis remplis ou colle les infos du certificat. Le logiciel mettra à jour la fiche.

2 — Coller un JSON de mise à jour

Utilise ce bloc quand je te fournis les données extraites des PDF. Le logiciel garde uniquement CIS Bouillante et ignore les dates plus anciennes.

3 — IA automatique Make / OpenAI Vision

Option avancée. Si tu n’as pas de webhook, ignore cette partie.

Sauvegarde / restauration

Après une mise à jour, clique sur Télécharger agents.json, puis remplace le fichier agents.json dans GitHub pour rendre la mise à jour permanente.
