PDF certificat médical

Pour l’instant, copie/colle le texte OCR du certificat ou les informations principales. La future automatisation Make récupérera le PDF, lira le certificat, vérifiera “CIS BOUILLANTE”, puis mettra à jour la fiche.

Sauvegarde / Import

Connexion Google Sheet

Quand Make sera configuré, renseigne l’URL CSV publiée dans config.js. L’application chargera alors automatiquement les données en ligne.

window.APP_CONFIG = { DATA_URL: "COLLER_ICI_URL_CSV_GOOGLE_SHEET" };

Données stockées localement sur l’appareil. Pour un usage réel, conserver seulement l’avis administratif d’aptitude, les restrictions nécessaires au service et les dates. Ne pas stocker de diagnostic médical.
